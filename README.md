
# Text Analysis and Classification using ML

## Objective 1: Resume Information Extractor
A rule-based Python script using regular expressions to extract
email addresses and phone numbers from resume text and store
them in a structured JSON file.

### Files
- extractor.py
- sample_resumes.txt
- output.json

---

## Objective 2: Sentiment Classification
Sentiment analysis performed on the IMDB movie reviews dataset.

### Dataset
IMDB Movie Reviews Dataset  
Source: https://ai.stanford.edu/~amaas/data/sentiment/

(Note: Dataset not uploaded due to GitHub file size limits)

### Techniques Used
- Text cleaning and preprocessing
- TF-IDF vectorization
- Machine Learning models:
  - Naive Bayes
  - Support Vector Machine (SVM)
  - Random Forest
- Evaluation using Accuracy, Precision, Recall, F1-score
- Confusion Matrix visualization
